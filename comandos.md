# Mis Notas de Linux 🐧

## Comandos básicos:
* `pwd`: ¿Dónde estoy? (Print Working Directory).
* `ls -la`: Ver todo lo que hay en la carpeta (incluso lo oculto).
* `mkdir [nombre]`: Crear una carpeta nueva.
* `cd [nombre]`: Entrar en una carpeta.
* `touch [archivo]`: Crear un archivo vacío.
* `cat [archivo]`: Leer el contenido de un archivo sin editarlo.

## Git (El Guardado):
* `git add .`: Preparar los cambios.
* `git commit -m "Mensaje"`: Etiquetar el cambio.
* `git push`: Subir a la nube.
